# AGB Photos CDN

Public CDN for AGB product photos used in marketplace listings (Ozon, Wildberries, Я.Маркет).

## URL pattern

```
https://raw.githubusercontent.com/bychenkovsv34-ops/agb-photos/main/<category>/<sku>/<slot>.<ext>
```

Example:
```
https://raw.githubusercontent.com/bychenkovsv34-ops/agb-photos/main/puhoviki/3507-TEMNZEL-40/01.jpg
```
